# docker-education-repo

### Сборка проекта
После того как в docker-compose.yml внесены все необходимые инструкции, проект нужно собрать.
```bash
docker-compose build
```
### Запуск проекта
Теперь, когда проект собран, пришло время его запустить.
```bash
docker-compose up
```
### Остановка проекта
```bash
docker-compose down
```
